# AreaPredictions
Area and Sub Area Predictions based given Summary and Description. 


This model is capable of predicting different Areas and subareas for a given a Summary or Description to content based text. This is a deep neural network model based on Bidirectional LSTM with Glove vectors embedded to tokenized text.

Data extracted from CRM internal website and didn't published here due to internally policy. Data consists of Text either Summary or Description and Area or Sub Area categeory.  

Achieved an accuracy of range 98-99% on test set.
